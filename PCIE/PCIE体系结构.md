# PCIE体系结构

## 第一章：概述

### 1.基本知识

#### 1.1PCI总线的基本知识

**主要目的**：为了连接外部设备。PCI总线，系统总线和处理器体系结构之间依然存在着紧密的联系。

##### 1.1.1PCI总线空间预处理器空间隔离

1.PCI设备具有独立的地址空间，该空间与存储器地址空间通过HOST主桥隔离。处理器需要通过**HOST主桥**才能访问PCI设备，PCI设备需要通过HOST主桥才能访问主存储器。

2.在HOST主桥中含有许多缓冲，使得处理器总线与PCI总线工作在各自的时钟频率中互不干扰。

３.HOST主桥的存在使得PCI设备核处理器可以方便地共享主存储器资源

##### 1.1.2.可扩展性

1.在**PCI 总线**中，**HOST主桥**可以直接推出一条**PCI总线**,该总线可以通过**PCI桥**扩展出一系列**PCI总线**，一个**PCI总线**树上最多挂载256个PCI设备

2.**同一条PCI总线**上的设备可以直接通信，在**同一颗PCI总线树**上的设备也可以直接通信，但需要通过**PCI桥**进行数据转发。

3.**PCI桥**是PCI总线的一个重要组成部分，他的存在时得PCI总线极具扩展性。

4.**PCI 桥**可以管理这个**PCI 总线子树**，在PCI桥的两端分别连接了两条总线，两条总线间的通信需要通过**PCI桥**进行。

**HOST**主桥：连接了CPU/内存世界和PCI总线世界，系统的第一条总线（**BUS0**）由它产生。

**总线层次**：

对**PCI桥1**来说，Bus0为**上游总线**，Bus1为**下游总线**

对**PCI桥2**来说， BUS1为**上游总线**， Bus2为**下游总线**

通信路径：CPU -> HOST主桥 -> Bus 0 -> PCI桥1 -> Bus 1 -> PCI桥2 -> Bus 2 -> 设备D

每一个桥设备都会检查地址，决定是否将数据包转发到自己的下游总线。

##### 1.1.3 PCI 设备

在PCI总线中有三类设备：**PCI**主设备，**PCI**从设备和桥设备。

**PCI从设备**：被动接收来自HOST主桥或其他PCI设备的读写请求

**PCI主设备**：可通过总线仲裁过得PCI总线的使用权，主动发起读写请求

**桥设备**：管理下游PCI总线，并转发上下游总线之间的总线事务

在PCI总线中，HOST主桥是一个特殊的PCI设备，该设备可以获取PCI总线控制权访问PCI设备，也可以被PCI设备访问

##### 1.1.4 HOST处理器

PCI总线规定在同一时刻，在一棵PCI总线树上有且只有一个HOST处理器

PCI总线树的实际管理者是HOST主桥，而不是HOST处理器

##### 1.1.5 PCI总线的负载

PCI 总线能挂接的负载与总线频率相关， 其中总线频率越高， 能挂接的负载越少。

#### 1.2 PCI总线的信号定义

PCI总线是一条共享总线，在一条PCI总线上可以挂接多个PCI设备。这些PCI设备通过一些系列信号与PCI总线相连，这些信号由地址/数据信号，控制信号，仲裁信号，中断信号等。

##### 1.2.1 地址和数据信号

在PCI总线中，与地址和数据相关的信号如下所示。

1. AD[31:0]信号

   PCI总线复用地址与数据信号。

2. PAR信号

   PCI总线使用奇偶校验机制，保证地址和数据在进行数据传递时的正确性。

3. C/BE[3:0]#信号

   PCI总线复用命令与字节选通引脚。

##### 1.2.2 接口控制信号

在PCI总线中，接口控制信号的主要作用是保证数据的正常传递，并根据PCI主从设备的状态，暂停，终止或正常完成当前总线事务

1. FRAME#信号

   该信号指示一个PCI总线事务的开始与结束

2. IRDY#信号

   该信号由PCI主设备（包括HOST主桥）驱动，该信号有效时表示PCI主设备的数据已准备完毕

3. TRDY#信号

   该信号由目标设备驱动，该信号有效时表示目标设备已经将数据准备完毕

4. STOP#信号

   该信号有效时表示目标设备请求主设备停止当前PCI总线事务。

5. IDSEL信号

   PCI总线在进行配置读写总线事务时，使用该信号选择PCI目标设备。

6. DEVSEL#信号

   该信号有效时表示PCI总线的目标设备准备好，该信号与TRDY#信号的不同之处在于该信号有效仅表示目标设备已经完成了地址译码。

7. LCOK#信号

   PCI主设备可以使用该信号，将目标设备的某个存储器或者I/O资源锁定，已禁止其他PCI主设备访问此资源，直到锁定这个资源的主设备将其释放。

##### 1.2.3 仲裁信号

PCI设备使用该组信号进行总线仲裁，并获得PCI总线的使用权。只有PCI主设备需要使用改组信号，而PCI从设备可以不使用总线仲裁信号。
